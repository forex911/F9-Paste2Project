Process

Three steps.
That's it.

No config files. No interactive prompts. Just paste and go.

01

Run the tool

Open your terminal. Type one command. f9 starts listening immediately — no flags, no arguments needed.

$ f9
02

Paste your structure

Copy any folder tree from GitHub, a README, an AI assistant, or your notes. Paste it directly into the terminal.

Ctrl+V → paste tree
03

Press CTRL+Z

Signal EOF. f9 parses indentation, detects folders vs files, and creates everything instantly.

CTRL+Z → project created
Capabilities

Everything you need.
Nothing you don't.

Built for speed. Engineered for precision.

📁

Auto-create Folders

Detects directory entries automatically based on trailing slashes and indentation depth.

📄

File Generation

Creates empty files with correct extensions exactly where they belong in the tree.

🌲

Tree Structure Support

Handles standard tree formats — │, ├──, └── — the same output you'd copy from any OS or tool.

⚡

Smart Indentation

Parses mixed indentation styles: spaces, tabs, or tree-drawing characters alike.

🪶

Lightweight & Fast

Zero dependencies. Runs in milliseconds. Binary under 1MB — fits on a USB drive.

🪟

Windows CLI Native

Built for Windows CMD and PowerShell. One installer batch file — no PATH gymnastics.

📋

Stdin-based Input

Reads directly from stdin. Composable with pipes, scripts, and automation workflows.

🔒

Offline & Private

No internet. No telemetry. No account. Your project structure stays on your machine.
