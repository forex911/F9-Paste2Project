How it works

Three steps. That's it.

No config files. No interactive prompts. Paste and go.

01

Run the tool

Open your terminal and type one command. f9 starts listening immediately — no flags or arguments needed.

$ f9
02

Paste your structure

Copy any folder tree from GitHub, a README, or an AI assistant and paste it directly into the terminal.

Ctrl+V → paste tree
03

Press CTRL+Z

Signal end-of-file. f9 parses indentation, detects folders and files, and creates everything instantly.

CTRL+Z → done
