Process

Three steps.
That's all.

No configuration files. No interactive prompts. Paste and go.

Step 01

Run the tool

Open your terminal and type one command. f9 starts listening immediately — no flags, no arguments required.

$ f9
Step 02

Paste your structure

Copy any folder tree from GitHub, a README, an AI assistant, or your notes. Paste it directly into the terminal.

Ctrl+V → paste tree
Step 03

Press CTRL+Z

Signal end-of-file. f9 parses indentation, detects folders and files, and creates everything instantly.

CTRL+Z → project created
Capabilities

Everything you need.
Nothing you don't.

Built for speed. Engineered for precision.

Directories

Auto-create Folders

Detects directory entries automatically based on trailing slashes and indentation depth.

Files

File Generation

Creates empty files with correct extensions exactly where they belong in the tree.

Parsing

Tree Structure Support

Handles standard tree formats — │, ├──, └── — the same output you'd copy from any OS or tool.

Intelligence

Smart Indentation

Parses mixed indentation styles: spaces, tabs, or tree-drawing characters alike.

Performance

Lightweight & Fast

Zero dependencies. Runs in milliseconds. Binary under 1MB — fits on a USB drive.

Platform

Windows CLI Native

Built for Windows CMD and PowerShell. One installer batch file — no PATH gymnastics.

I/O

Stdin-based Input

Reads directly from stdin. Composable with pipes, scripts, and automation workflows.

Privacy

Offline & Private

No internet. No telemetry. No account. Your project structure stays on your machine.

Installation

Up and running
in 60 seconds.

Three commands. One batch file. You're done.

Windows CMD / PowerShell
# 1. Clone the repository
git clone https://github.com/forex911/f9-paste2project.git
 
# 2. Navigate into the folder
cd f9
 
# 3. Run the installer
f9_Installer.bat
 
# Done — f9 is now available globally
